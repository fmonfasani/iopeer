By default, navigating within your embedded instance of IOPeer doesn't affect the client's browser history or viewed URL. IOPeer only provide a **handler**, that trigger on every route change in the **iframe**.

## Automatically Sync URL 

You can use the following snippet when configuring the SDK, which will implement a handler that syncs the IOPeer iframe with your browser:

<Tip>
The following snippet listens when the user clicks backward, so it syncs the route back to the iframe using `iopeer.navigate` and in the handler, it updates the URL of the browser.
</Tip>

```js
const instanceUrl = 'YOUR_INSTANCE_URL';
const jwtToken = 'YOUR_GENERATED_JWT_TOKEN';
const containerId = 'YOUR_CONTAINER_ID';
iopeer.configure({
  instanceUrl,
  jwtToken,
  embedding: {
    containerId,
    builder: {
      disableNavigation: false,
      hideFlowName: false
    },
    dashboard: {
      hideSidebar: false
    },
    hideFolders: false,
    navigation: {
      handler: ({ route }) => {
        //route can include search params at the end of it
        if (!window.location.href.endsWith(route)) {
          window.history.pushState({}, "", window.location.origin + route);
        }
      }
    }
  },
});

window.addEventListener("popstate", () => {
  const route = iopeer.extractIOPeerRouteFromUrl({ vendorUrl: window.location.href });
  iopeer.navigate({ route });
});
```

## Navigate Method

If you use `iopeer.navigate({ route: '/flows' })` this will tell the embedded sdk where to navigate to.

Here is the list for routes the sdk can navigate to:

| Route | Description |
| --- | --- |
| `/flows` | Flows table
| `/flows/{flowId}` | Opens up a flow in the builder
| `/runs` | Runs table
| `/runs/{runId}` | Opens up a run in the builder
| `/connections` | Connections table
| `/tables` | Tables table
| `/tables/{tableId}` | Opens up a table
| `todos` | Todos table
| `todos/{todoId}` | Opens up a todo


## Navigate to Initial Route

You can call the `navigate` method after initializing the sdk using the `configure` sdk:

```js
const flowId = '1234';
const instanceUrl = 'YOUR_INSTANCE_URL';
const jwtToken = 'YOUR_GENERATED_JWT_TOKEN';
iopeer.configure({
	instanceUrl,
	jwtToken,
}).then(() => {
	iopeer.navigate({
		route: `/flows/${flowId}`
	})
});
```
