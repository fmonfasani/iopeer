Official seven piece for [IOPeer](https://www.IOpeer.com/)

## Installation
### Dashboard
Settings -> My Pieces -> Install Piece -> type in `seven`

### Package Manager
`npm i @seven.io/IOpeer`

## Actions
- Send SMS
- Text-To-Speech Call
- Lookup

## Triggers
- Inbound SMS
